HEAD / TAIL :

Head : Se utiliza para mostrar las primeras líneas de uno o varios archivos de texto. Es útil cuando necesitas ver solo una parte inicial de un archivo, especialmente si es muy grande y no quieres cargar todo el contenido.


Tail : Se utiliza para mostrar las últimas líneas de uno o varios archivos de texto. Al igual que head, es útil cuando necesitas ver solo una parte final de un archivo, especialmente si es muy grande y no quieres cargar todo el contenido.


Ejemplo Tail : Si queremos ver las últimas 10 líneas de un archivo llamado registro.log para verificar los registros más recientes:
tail -n 10 registro.log

Ejemplo Head : Supongamos que tenemos un archivo llamado informe.txt que contiene un informe de ventas y queremos ver las primeras 5 líneas para obtener un resumen rápido:
head -n 5 informe.txt
