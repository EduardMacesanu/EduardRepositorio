CHMOD :

Se utiliza para cambiar los permisos de acceso de archivos y directorios. Estos permisos controlan quién puede leer, escribir y ejecutar un archivo o directorio. El comando chmod permite al usuario cambiar estos permisos, tanto para el propietario del archivo como para los grupos de usuarios y otros usuarios.


Ejemplo : Imaginemos que tenemos un archivo llamado script.sh que queremos hacer ejecutable para todos los usuarios, y permitir que el propietario tenga permisos de lectura y escritura, pero sin permisos de ejecución:
chmod 644 script.sh
